The **starlight-rapidoc** plugin integrates [Rapidoc](https://rapidocweb.com/) into Starlight documentation sites, enabling the display of OpenAPI specifications directly within documentation pages.

## Features

The plugin extends Starlight's content schema to support OpenAPI documentation through frontmatter fields, allowing you to specify specification files, server URLs, and interaction options. It provides customizable theming with configurable color schemes for both light and dark modes, ensuring consistent visual integration with your documentation site.

## How It Works

The plugin overrides Starlight's `TwoColumnContent` component to conditionally render Rapidoc when an OpenAPI specification is present. Pages with OpenAPI specifications are automatically rendered using Rapidoc's interactive documentation viewer, while standard documentation pages maintain Starlight's default layout.

## Configuration

The plugin accepts optional configuration parameters for customizing the color scheme:

- `hexColorBgLight` - Background color for light mode
- `hexColorBgDark` - Background color for dark mode
- `hexColorFgLight` - Foreground color for light mode
- `hexColorFgDark` - Foreground color for dark mode

All color parameters accept hexadecimal color values and default to Starlight-compatible colors if not specified.
